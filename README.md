## Steps to Install - ğŸš€ Getting Started (Dev)
- Run the command below from the command line / terminal / command prompt.
- git clone https://github.com/systemsvanguard/actorsappvue.git  
- cd actorsappvue
- ensure your have Node & NPM pre-installed. Run commands 'node --version && npm -v'.
- npm install.  (This ensures all dependencies are installed).
- npm run dev 
- Runs on port 5173 ---> http://localhost:5173/  
- Change the web port as needed.
- Please also see the related & prerequisite project, the backend REST API, located at  https://actorsapi.ryanhunter.ca/ 
- See also the related React.js project at Git repo https://github.com/systemsvanguard/actorsappreact.git and site at https://actorsappreact.ryanhunter.ca/.
